🚀 HOW TO START:

Step 1: Read Documentation
  → Open IMPLEMENTATION_SUMMARY.md (5 minutes)
  → Understand what was built

Step 2: Run Installation
  → Windows: Double-click INSTALL.bat
  → Linux/Mac: Run bash INSTALL.sh
  → Creates /uploads/notes/ directory

Step 3: Import Database
  → Option A: Use phpMyAdmin
    1. http://localhost/phpmyadmin
    2. Click "Import" tab
    3. Select database/studyshare.sql
    4. Click "Go"

  → Option B: Use MySQL Command Line
    mysql -u root studyshare < database/studyshare.sql

Step 4: Test the System
  → Open http://localhost/STUDYSHARE-/index-test.html
  → Follow the testing checklist
  → Try student and lecturer portals
